**Dependencies**

## Running the project

Clone this repository :

```
git clone https://github.com/LilaGuill/stateofjs-frontend.git
cd stateofjs-frontend
```

Install packages :

```
npm install
```

When installation is complete, run the project with:

```
npm start
```
